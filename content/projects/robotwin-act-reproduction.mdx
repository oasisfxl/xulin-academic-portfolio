## Overview

This page is a living reproduction log for ACT-style imitation learning in RoboTwin. The goal is not only to reproduce headline performance, but also to document the fragile parts of the pipeline: dataset conventions, action normalization, training stability, and evaluation mismatch.

<Callout title="Writing status" type="note">
This is an initial public skeleton. Detailed plots, command logs, and videos can be added as the reproduction record matures.
</Callout>

## Reproduction Checklist

- Confirm task configuration and observation keys.
- Verify action chunk size and temporal alignment.
- Compare dataset statistics before training.
- Track validation rollouts, not only training loss.
- Save failure cases with short captions and timestamps.
